# Building Mobile App (Android/iOS)

This app is configured for **mobile platforms only** (Android & iOS).

## Build Commands

### Android APK (Release)
```bash
flutter build apk --release
```
Output: `build/app/outputs/flutter-apk/app-release.apk`

### Android App Bundle (for Play Store)
```bash
flutter build appbundle --release
```
Output: `build/app/outputs/bundle/release/app-release.aab`

### iOS (requires Mac)
```bash
flutter build ios --release
```

## App Configuration

- **Package Name**: `com.rpntechworld.medai`
- **App Name**: Med AI - Medicinal Plants
- **Version**: 1.0.0+1
- **Min Android SDK**: 21 (Android 5.0+)
- **Target Platforms**: Android & iOS only

## Notes

- This app is designed for mobile devices only
- Web, Windows, Linux, and macOS builds are not supported
- Optimized for touch interfaces and mobile screen sizes
- Requires internet connection for AI features
